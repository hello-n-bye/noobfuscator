# Noobfuscator 🐈
The world's worst open-sourced obfuscator, built in pure JavaScript.

## Features
1. Lua minification
2. Rename variables, function names & types → `local function` → `function`
3. String encryption → `print("Bonjour")` → `print("\x42\x6f\x6e\x6a\x6f\x75\x72")`
